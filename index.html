Text styling to be used

  • Headings and Top Header

    • The main page heading has an H1 tag
    • With a font-family value: Garamond
    • With a font-weight value: 900
    • Default color of fonts: Gray

  • Subheadings and Sub sections

    • Every sub heading enclosed in an H2 tag
    • Having a cornflower blue background by default
    • With default black color

  • Paragraphs

    • Each paragraph has a default font of Garamond
    • Default color: Black
    • Font Size: 18px

  • Buttons and colored Divs

    • Border radius applied to the top right and bottom left of each box-like div
    • Button could have other radius styling
    • Buttons and colored divs follow the defined color set
